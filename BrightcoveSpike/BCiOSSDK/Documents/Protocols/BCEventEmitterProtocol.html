BCEventEmitterProtocol Protocol Reference

Declared in BCEventEmitterProtocol.h

Overview

Defines the interface for an event emitter that can receive and emit events.

Tasks

Adding and Removing Listeners

  • – on:callBlock:

    Allows an object to add itself as a listener for a type of event by passing in a block to be invoked when an event of that type is emitted.

         required method
  • – once:callBlock:

    Allows an object to add itself as a listener for a type of event by passing in a block to be invoked when an event of that type is emitted. Immediately before invoking the block, the listener will be automatically removed, however an opaque object is still returned so the listener may be removed before it executes.

         required method
  • – off:removeListener:

    Removes a listener from the list of listeners for the given type. If no previously added listener matches the parameters, calling this method has no effect.

         required method

Emitting Events

  • – emit:withDetails:thenCallBlock:

Request-Reply Flow

  • – request:withDetails:responseBlock:

    Emits an event of the specified type with the associated details. The emitter will listen for responses to this event and call the provided responseBlock when the first response is encountered. Additional responses will be ignored.

         required method
  • – respond:emit:withDetails:

    Emits an event of the specified type as a response to a previous event.

         required method

request:withDetails:responseBlock:

Emits an event of the specified type with the associated details. The emitter will listen for responses to this event and call the provided responseBlock when the first response is encountered. Additional responses will be ignored.

- (void)request:(NSString *)type withDetails:(NSDictionary *)details responseBlock:(BCCallbackBlock)responseBlock

Parameters

type

The type of Event to emit.

details

Dictionary of additional details relevant to this event. Must not be nil since this is used to determine the matching response Event.

responseBlock

The block of code to invoke when a response Event is received.
